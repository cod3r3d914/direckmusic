# DireckMusic

Official static website for DireckMusic, independent Raleigh hip-hop by Cody.

## Local Preview

Open `index.html` directly in a browser, or serve the folder with any static server.

## Files

- `index.html` - page content, SEO tags, structured data, and social links
- `styles.css` - responsive visual system and layout
- `script.js` - mobile nav and animated waveform
- `assets/direck-avatar.jpg` - public artist avatar from SoundCloud
